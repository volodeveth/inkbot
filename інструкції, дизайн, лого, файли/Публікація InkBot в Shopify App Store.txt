================================================================================
Етап 3: Подача на Review
================================================================================

Що перевіряє Shopify:
1. Функціональність — чи працює як описано
2. UI/UX — чи дотримується Polaris guidelines
3. Безпека — OAuth, API keys, data handling
4. Performance — швидкість, error handling
5. Billing — коректність charges
6. Content — screenshots відповідають реальному UI

Термін review: 5-10 робочих днів (іноді до 3 тижнів)
