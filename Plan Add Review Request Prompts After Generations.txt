│                                                                                                                                                                                                                 │
│ Show a review request banner after the 1st, 5th, and 10th lifetime generation. Once the user clicks "Leave a Review" or dismisses the prompt, it never appears again.                                           │
│                                                                                                                                                                                                                 │
│ How it works                                                                                                                                                                                                    │
│                                                                                                                                                                                                                 │
│ - Track lifetime generation count via getGenerationStats().totalGenerations (already exists)                                                                                                                    │
│ - Add reviewLeft Boolean @default(false) field to Shop model — set to true when user clicks review link or dismisses                                                                                            │
│ - In app.generate.tsx and app.bulk.tsx actions: after successful generation, check if totalGenerations is 1, 5, or 10 AND reviewLeft is false → return showReviewBanner: true in response                       │
│ - In the UI: show a Polaris Banner with review link + dismiss button                                                                                                                                            │
│ - Dismiss action sets reviewLeft = true on Shop                                                                                                                                                                 │
